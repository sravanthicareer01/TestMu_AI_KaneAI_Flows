# TestMu_KaneAI_Flows

**Overview ** 
This repository contains my QA automation work using TestMu KaneAI.  
It focuses on UI testing flows, with documentation and demo video links included.  
The aim is to show practical automation skills and how AI can support testing.

**Skills Covered in KaneAI UI Testing  **
- UI automation basics such as login, navigation, and checkout flows  
- Locator handling including selectors and nested locators in Playwright  
- Assertions to validate expected outcomes  
- Error handling and reporting  
- Designing end-to-end test flows  
- Writing clear documentation for test steps and results  
- Connecting UI flows with API and DB validations as a foundation for future work
